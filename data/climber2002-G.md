# Cache derivativeCount in a local variable when looping
`derivativeCount` is a storage variable. When looping it's more gas efficient to cache it in a local variable.

Change
```solidity
for (uint256 i = 0; i < derivativeCount; i++)
```

to
```solidity
uint256 localDerivativeCount = derivativeCount;
for (uint256 i = 0; i < localDerivativeCount; i++)
```
